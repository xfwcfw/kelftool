# kelftool
An open-source utility for decrypt, encrypt and sign PS2 KELF and PSX KELF files.

#### You need to bring your own keys.

Place them at your home directory (%USERPROFILE%) in "PS2KEYS.dat" file as a 'KEY=HEX_VALUE' pair.
